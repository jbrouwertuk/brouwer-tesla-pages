Tesla OAuth-callback

Tesla heeft je doorgestuurd met een autorisatie-code. Kopieer die code en plak hem in de admin-pagina van het Syntess-dashboard.

Wachten op redirect vanuit Tesla…

Als je hier direct terechtkwam zonder via Tesla in te loggen, start de koppeling opnieuw via de admin-pagina.

Volgende stap:
Ga terug naar http://<dashboard-ip>:8080/admin/instellingen → Tesla-tab → plak de code in het daarvoor bestemde veld → Opslaan.

De code is slechts korte tijd geldig (± 5 minuten).
